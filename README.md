## Description

A quick and dirty project to have a CoreDNS and simple API to update zone records (with NestJS).
It supports docker and a API key for "security".  

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
